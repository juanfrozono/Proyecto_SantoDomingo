# Proyecto_SantoDomingo
#Participantes
1-Admin => publica comunicados ,eventos y notas.
2-Profesores => carga y edita notas de los estudiantes. 
3-Estudiantes => ven sus notas ,lee comunicados(información). 

#Reglas base
-En la parte de notas podría ser: 
a-Solo el docente puede cargar notas. 
b-Un estudiante solo puede ver sus propias notas(no las de los demás). 
c-La nota debe ser un número entre 0 y 20. 
d-Si el Docente no ha ingresado la nota ,mostrar “Pendiente”. 

-Ahora en la parte de difusión de información: 
A-Los comunicados serán de manera general 
B-Si hay algún evento, reunión o algo puede tener título,fecha,hora,lugar,motivo o descripción  
C-Si algún evento es hoy,mostrarlo como fijado 

-Sobre el acceso: 
Los estudiantes deben iniciar sesión para ver sus notas.Sin login,solo puede ver comunicados públicos
